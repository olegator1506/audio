==================== Инициализация Adau ===================
При старте в ADAU записываются данные инициализации формируемые из проекта SigmaStudio
1. В SigmaStudio открываем проект и выполняем в тулбаре "export system files" 
сохраняем в отдельную папку, например под именем sample1
2. копируем файлы заголовочные файлы (*.h) в папку includes, перименовываем главный заголовочный файл
*_IC_1.h в adau_project.h 
==================== Работа с SigmaStudio
Для организации передачи данных из SigmaStudio в чип ADAU176x делаем следующее:
на вкладке Hardware Comfiguration / Config выбираем нужный чип (в нашем случае ADAU176x) и в 
из списка Communication Channels выбираем TCPIPADAU1761, соединяем их.
На модуле TCPIP в контекстном меню выбираем Show TCPIP Settings, вводим IP адрес нашего модуля
и жмем кнопку OpenConnction
! Настройка IP адреса не сохраняется, поэтому адрес нужно вводить при каждом запуске SigmaStudio 

Программа принимает запросы от SigmaStudio и осуществляет запрашиваемые операции чтения/записи

Формат пакета
Байт 0 - команда 0x0a - чтение из ADAU, иначе запись
Байт 1 : 2 total_len (байт 1 - старший) 
Байт 3 :
Байт 4 : 5 len (байт 4 - старший) 
Байт 6 : 7 addr (байт 6 - старший) 

=================== Запись в ADAU ==================
Байт 0:1 адрес (байт 0 старший)
Байт 2 : len+1 данные

===================  Чтение из ADAU ==============
Запрос
CMD = 0x0a
Ответ
Byte[0] = 0x0b
Byte[1-2] = len+4 
Byte[3] = 0 (признак успешного чтения)
Byte[4...] считаные данные
В ADAU передается 2 байта адреса

После передачи адреса принимаем данные в количестве len байт
